# FishMach 自动钓鱼机

- 作者: 羽学
- 出处: TShock官方群816771079
- 这是一个Tshock服务器插件，主要用于：使用/afm 指令指定一个箱子作为具备移动物品动画的自动钓鱼机,支持自定义渔获

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /afm set | afm.use | 选择箱子设为钓鱼机 |
| /afm save | afm.use | 添加/移除选择的钓鱼机 |
| /afm exc | afm.use | 添加/移除手持排除物品 |
| /afm list | afm.use | 列出所有机器 |
| /afm info | afm.use | 获取钓鱼机信息 |
| /afm loot | afm.use | 列出自定义渔获 |
| /afm cond | afm.admin | 查看自定渔获条件 |
| /afm item | afm.admin | 修改自定义渔获 |
| /afm reset | afm.admin | 重置插件数据 |
| /reload | tshock.cfg.reload | 重新加载配置文件 |

## 使用方法
- 在鱼池放个箱子,使用/afm set 镐击一下箱子，
- 使用/afm save 保存当前钓鱼机
- 给箱子放入鱼竿和鱼饵
- 挖掉箱子可自动销毁钓鱼机

```
v1.0.1
新增功能：自定义额外渔获(对应指令loot,cond,item)
新增排除指令：/afm exc
- 需在游戏内用s指令敲击自钓箱,仅拥有者和管理可用
- 根据手持物品来决定,存在则移除,不在则添加
性能优化：渐进式搜索（从2格逐步扩大），优先命中钓鱼箱，减少无效扫描。
动态查找物品：鱼竿、鱼饵、加成物品（宝箱药水、钓具箱等）不再存储，改为实时扫描附近箱子，灵活生效。
精准环境判定：修复环境匣无法掉落的问题，现在丛林、雪原、沙漠等生物群落均能正确钓出对应板条匣与任务鱼。
智能物品存放：优先放入钓鱼箱，满则自动寻找最近的其他箱子，并播放原版转移动画。
宝箱概率可调：新增配置项 宝箱概率加成，支持自定义板条箱掉落率；宝箱药水效果同步生效。
箱子变动简化：只有自定义加成物品变动时才保存数据，鱼竿、鱼饵变动只提示，不再误清机器状态。

v1.0.0
实现让箱子具备移动物品动画的自动钓鱼机

```
