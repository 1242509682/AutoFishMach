# FishMach 自动钓鱼机

- 作者: 羽学
- 出处: TShock官方群816771079
- 这是一个Tshock服务器插件，主要用于：
- 使用/afm 指令指定一个箱子作为具备移动物品动画的自动钓鱼机,支持自定义渔获物品与怪物

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /afm set | afm.use | 将箱子设为钓鱼机 |
| /afm save | afm.use | 同步更新钓鱼机 |
| /afm tp | afm.use | 传送到指定钓鱼机 |
| /afm out | afm.use | 批量设置传输箱 |
| /afm exc | afm.use | 批量修改排除物品表 |
| /afm info | afm.use | 获取钓鱼机信息 |
| /afm list | afm.use | 列出所有钓鱼机 |
| /afm loot | afm.use | 列出自定义渔获 |
| /afm solo | afm.use | 修改禁钓已有怪物模式 |
| /afm cond | afm.admin | 查看自定渔获条件 |
| /afm npc | afm.admin | 修改自定渔获怪物 |
| /afm item | afm.admin | 修改自定渔获物品 |
| /afm reset | afm.admin | 重置插件数据 |
| /reload | tshock.cfg.reload | 重新加载配置文件 |

## 使用方法
- 给箱子放入鱼竿和鱼饵,使用/afm s指令
- 如果开启【需要电路】,则需要连上电线与计时器
- 挖掉箱子可自动销毁钓鱼机与传输箱
- [可选] 在钓鱼机附近使用/afm o 离开区域打开其他箱子作为传输箱
- [传输箱] 支持跨区域转移物品(允许单机多箱 与 多机共用1箱)
- 重置服务器时使用:/afm reset
- 挖掉箱子自动销毁钓鱼机与传输箱

- 动态查找物品：鱼竿、鱼饵、加成物品（宝箱药水、钓具箱等）不再存储，改为实时扫描附近箱子，灵活生效。
- 精准环境判定：修复环境匣无法掉落的问题，现在丛林、雪原、沙漠等生物群落均能正确钓出对应板条匣与任务鱼。
- 智能物品存放：优先放入钓鱼箱，满则自动寻找最近的其他箱子，并播放原版转移动画。
- 宝箱概率可调：新增配置项“宝箱概率加成”，支持自定义板条箱掉落率；宝箱药水效果同步生效。
- 箱子变动简化：只有自定义加成物品变动时才保存数据，鱼竿、鱼饵变动只提示，不再误清机器状态。
