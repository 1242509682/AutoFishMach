# FishMach 自动钓鱼机

- 作者: 羽学
- 出处: TShock官方群816771079
- 这是一个Tshock服务器插件，主要用于：使用/afm 指令指定一个箱子作为具备移动物品动画的自动钓鱼机

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /afm set | afm.use | 选择箱子设为钓鱼机 |
| /afm save | afm.use | 添加/移除选择的钓鱼机 |
| /afm list | afm.use | 列出所有机器 |
| /afm info | afm.use | 获取钓鱼机信息 |
| /afm reset | afm.admin | 重置插件数据 |
| /reload | tshock.cfg.reload | 重新加载配置文件 |

## 更新日志
```
v1.0.0
实现让箱子具备移动物品动画的自动钓鱼机
使用方法：
在鱼池放个箱子,使用/afm set 镐击一下箱子，
使用/afm save 保存当前钓鱼机
给箱子放入鱼竿和鱼饵
当钓鱼机失效时,需重新放入鱼竿(乱动鱼饵和鱼竿引起的)
挖掉箱子可自动销毁钓鱼机
在钓鱼机箱左边放其他箱子,会优先放入左边的箱子
已知BUG:暂时无法钓环境匣子与板条箱
```
