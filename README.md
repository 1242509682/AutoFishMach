# FishMach 自动钓鱼机

- 作者: 羽学
- 出处: TShock官方群816771079
- 这是一个Tshock服务器插件，主要用于：
- 使用/afm 指令指定一个箱子作为具备移动物品动画的自动钓鱼机,支持自定义渔获物品与怪物

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /afm set | afm.use | 打开箱子设为钓鱼机 |
| /afm exc | afm.use | 打开箱子添加/移除手上排除物品 |
| /afm info | afm.use | 打开箱子获取钓鱼机信息 |
| /afm list | afm.use | 列出所有自钓机 |
| /afm loot | afm.use | 列出自定义渔获 |
| /afm cond | afm.admin | 查看自定渔获条件 |
| /afm npc | afm.admin | 修改自定渔获怪物 |
| /afm item | afm.admin | 修改自定渔获物品 |
| /afm reset | afm.admin | 重置插件数据 |
| /reload | tshock.cfg.reload | 重新加载配置文件 |

## 使用方法
- 放个箱子,使用/afm s指令,再打开箱子
- 给箱子放入鱼竿和鱼饵
- 如果开启【需要电路】,则需要连上电线与计时器
- 挖掉箱子可自动销毁钓鱼机
